# Установка
```
 git clone git@github.com:ctrlVadim/snake.git
 poetry install
 python3 __main__.py
```

# Запуск
```
 python3 __main__.py
```

# Управление

| Клавиша        | Действие                |
|----------------|-------------------------|
| ⬅ `Left`       | Движение влево          |
| ➡ `Right`      | Движение вправо         |
| ⬆ `Up`         | Движение вверх          |
| ⬇ `Down`       | Движение вниз           |
| ␣ `Space`      | Перезапуск игры         |
| ⎋ `Escape`     | Пауза / продолжить игру |
