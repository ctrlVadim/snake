# Установка
```
    git clone git@github.com:ctrlVadim/snake.git
    cd snake
    python3 -m venv venv
    poetry install
```

# Запуск
```
 python3 __main__.py
```

# Управление

| Клавиша        | Действие                |
|----------------|-------------------------|
| ⬅ `Left`       | Движение влево          |
| ➡ `Right`      | Движение вправо         |
| ⬆ `Up`         | Движение вверх          |
| ⬇ `Down`       | Движение вниз           |
| ␣ `Space`      | Перезапуск игры         |
| ⎋ `Escape`     | Пауза / продолжить игру |
